Node-obf

Javascript code obfuscator for NodeJS

Installation

To install node-obf, use npm:

$ npm install node-obf

Usage

require('node-obf').obfuscate(global variable, code);

Example:

var obf = require('node-obf');
obf.obfuscate('$$', 'alert("hello world!")');
